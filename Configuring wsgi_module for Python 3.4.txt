Configuring wsgi_module for Python 3.4


Ubuntu 14.04 LTS comes with mod_wsgi 3.4.
We need to use mod_wsgi version 4.2+ for Python 3.4.

1) Remove problematic package and install dependency
    sudo apt-get remove libapache2-mod-wsgi-py3
    sudo apt-get install apache2-dev

2) Install mod_wsgi
    pip3 install mod_wsgi

3) Install into Apache
    /usr/local/bin/mod_wsgi-express install-module

4) Create files
    /etc/apache2/mods-available/wsgi_express.conf
    /etc/apache2/mods-available/wsgi_express.load

5) Content of wsgi_express.load
LoadModule wsgi_module /usr/lib/apache2/modules/mod_wsgi-py34.cpython-34m.so

6) wsgi_express.conf is left empty

7) Enable the module and restart Apache:
    a2enmod wsgi_express
    service apache2 restart

8) [optional]
    a2dismod wsgi
    Delete wsgi.conf and wsgi.load from /etc/apache2/mods-available
    Delete corresp. links from /etc/apache2/mods-enabled
